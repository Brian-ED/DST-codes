## URL API ("permalinks")

After clicking [<kbd>#</kbd>](https://Brian-ED.github.io/DST-codes?q=42) simply copy the current address from your browser's address bar to share the current query. Note that your current theme is intentionally not included, to avoid blinding the recipient. Note that the result for such a URL isn't permanent; it depends on `DST-codes`'s content at the time of usage.

### Manual construction

After [Brian-ED.github.io/DST-codes](https://Brian-ED.github.io/DST-codes/) you can add `?` and the following options, separated by `&` if you want both a theme and a query:

- [<code>w</code>](https://Brian-ED.github.io/dst-codes?w)‚ÄÉselect **w**hite theme
- [<code>b</code>](https://Brian-ED.github.io/dst-codes?b)‚ÄÉselect **b**lack theme
- [<code>q=<i>query</i></code>](https://Brian-ED.github.io/dst-codes?q=query)‚ÄÉprepopulate the _Tell me about_ **q**uery field

For example, for a white interface looking for "find", use [Brian-ED.github.io/dst-codes?w&q=find](https://Brian-ED.github.io/dst-codes?w&q=find)
